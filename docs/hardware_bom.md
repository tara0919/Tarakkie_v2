- [ ] **マイコン**: Seeed XIAO nRF52840 (Ble) x2
- [ ] **IOエキスパンダ (入力拡張IC)**: MCP23017
    - [ ] **DIP版 (足つき)** x2 (ブレッドボード試作用) - [秋月電子: MCP23017](https://akizukidenshi.com/catalog/g/g109486/)
        - *I2C通信のためにプルアップ抵抗(4.7kΩ程度)がSDA, SCLラインに各1本必要です。*
    - [ ] **SSOP/SOIC版 (表面実装)** x2 (本番PCB用)
- [ ] **モジュール接続コネクタ**:
    - [ ] ポゴピン (スプリングピン): 10ピンブロック (2.54mmピッチ等)
    - [ ] マグネット: ネオジム磁石 (極性に注意して配置)
- [ ] **電源周り**:
    - [ ] バッテリー: 3.7V リチウムポリマー電池 (ケースに入る薄型)
    - [ ] 電源スイッチ: スライドスイッチ (PCM12等)
- [ ] **共通部品**:
    - [ ] I/O拡張IC: MCP23017 (I2C接続、メインマトリクス用)
    - [ ] **モジュール接続用コネクタ**: 12ピン マグネットポゴピンコネクタ
    - [ ] プッシュスイッチ: リセット用 (Xiao BLEのRSTピン)
- [ ] **キー用部品**:
    - [ ] スイッチ: **Kailh Choc V1 または V2** (ロープロファイル)
    - [ ] ダイオード: 1N4148 (PCBなら表面実装用、手配線ならリード付き)
    - [ ] ソケット: Kailh Choc用 ホットスワップソケット

### モジュール3 (アナログ + エンコーダー)
- [ ] **アナログジョイスティック**: [g114675 (RKJXY)](https://akizukidenshi.com/catalog/g/g114675/) - 指で倒すタイプ（ゲーム機に近い）

- [ ] **表面実装用ロータリーエンコーダー**: [g114936](https://akizukidenshi.com/catalog/g/g114936/)
- [ ] メカニカルスイッチ x 1
    - *ピン数*: 9-10ピン (VCC, GND, ADCx2, StickSWx1, Enc x3, Key x1) -> **OK**

### モジュール4 (トラックパッド + エンコーダー)
- [ ] **トラックパッドモジュール**: 以下から選択
    - **Cirque GlidePoint (35mm)**: [TM035035](https://www.mouser.jp/ProductDetail/Cirque/TM035035-2024-001?qs=wd5RIQLrsJjQWLerbkGS5g%3D%3D)
    - **変換基板 (重要)**: FPC(12ピン 0.5mmピッチ)を普通のピンに変換するもの
        - [Beekeeb (日本サイト)](https://beekeeb.jp/): 「Cirque GlidePoint Trackpad breakout」がセットで売られています。
        - **汎用品**: 秋月などで「0.5mmピッチ 12ピン FPC変換基板」を探す（ハンダ付けの難易度が少し高いです）。
- [ ] **表面実装用ロータリーエンコーダー**: [g114936](https://akizukidenshi.com/catalog/g/g114936/)
    - *ピン数*: 8ピン (VCC, GND, I2Cx2, INT, Enc x3) -> **OK (12ピンに余裕)**

## モジュール5: 乾電池駆動モジュール
- [ ] **電池ボックス**: 単3または単4電池用
- [ ] **昇圧コンバータ (Step-up)**: 1.5V/3.0V → 3.3V/5V
    - *ピン数*: 4ピン (VCC, GND, BAT_IN, 3.3V_OUT) -> **OK**
    - *重要*: Xiaoの3.3Vピンに入れるか、5Vピンに入れるか要検討。一次電池(充電不可)をリポ用端子(BAT)に入れるのは**厳禁**です（USB接続時に爆発のリスクあり）。

## モジュール6: 液晶モニター
- [ ] **ディスプレイ**: **Nice!View** (Memory LCD) または **OLED (SSD1306)**
    - *推奨*: Nice!View (省電力で常時点灯可能、ZMKで標準サポート)
    - *接続*: I2CまたはSPI (Nice!ViewはCS, MOSI, CLK + 電源の5本必要)
