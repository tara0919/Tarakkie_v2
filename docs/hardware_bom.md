# ハードウェア部品リスト (BOM) - Tarakkie_v2

## ベースボード (左右分割キーボード本体)
- [ ] **マイコン**: Seeed XIAO nRF52840 (Ble) x2
- [ ] **IOエキスパンダ (入力拡張IC)**: MCP23017
    - [ ] **DIP版 (足つき)** x2 (ブレッドボード試作用) - [秋月電子: MCP23017](https://akizukidenshi.com/catalog/g/g109486/)
        - *I2C通信のためにプルアップ抵抗(4.7kΩ程度)がSDA, SCLラインに各1本必要です。*
    - [ ] **SSOP/SOIC版 (表面実装)** x2 (本番PCB用)
- [ ] **モジュール接続コネクタ**:
    - [ ] ポゴピン (スプリングピン): 10ピンブロック (2.54mmピッチ等)
    - [ ] マグネット: ネオジム磁石 (極性に注意して配置)
- [ ] **電源周り**:
    - [ ] バッテリー: 3.7V リチウムポリマー電池 (ケースに入る薄型)
    - [ ] 電源スイッチ: スライドスイッチ (PCM12等)
- [ ] **キー用部品**:
    - [ ] スイッチ: **Kailh Choc V1 または V2** (ロープロファイル)
    - [ ] ダイオード: 1N4148 (PCBなら表面実装用、手配線ならリード付き)
    - [ ] ソケット: Kailh Choc用 ホットスワップソケット

## モジュール1: トラックボール + エンコーダ
- [ ] **トラックボールセンサー**: PMW3610
    - *推奨*: **PMW3610 ブレイクアウト基板** (自作キーボードショップ等で販売されているもの)
        - *理由*: センサー単体は配線が非常に難しく、レンズ等の光学部品に厳密な位置合わせが必要なため、手配線や自作回路での実装は非常に困難です。
- [ ] **ロータリーエンコーダ**: EC11 (あれば低背のAlps EC11など)

## モジュール3: アナログパッド
- [ ] **アナログジョイスティック**: **PSP-2000/3000スタイル スライドパッド**
    - *特徴*: 非常に薄く、Chocキーとの相性が良い。

## モジュール4: トラックパッド
- [ ] **タッチセンサー**: **四角い静電容量式タッチモジュール**
    - *候補*: Azoteq IQS5xxシリーズ搭載の四角いトラックパッド、または汎用品
    - *条件*: I2C接続ができること

## モジュール5: 乾電池駆動モジュール
- [ ] **電池ボックス**: 単3または単4電池用
- [ ] **昇圧コンバータ (Step-up)**: 1.5V/3.0V → 3.3V/5V
    - *重要*: Xiaoの3.3Vピンに入れるか、5Vピンに入れるか要検討。一次電池(充電不可)をリポ用端子(BAT)に入れるのは**厳禁**です（USB接続時に爆発のリスクあり）。

## モジュール6: 液晶モニター
- [ ] **ディスプレイ**: **Nice!View** (Memory LCD) または **OLED (SSD1306)**
    - *推奨*: Nice!View (省電力で常時点灯可能、ZMKで標準サポート)
    - *接続*: I2CまたはSPI (Nice!ViewはCS, MOSI, CLK + 電源の5本必要)
